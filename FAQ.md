# FAQ

## Is this project legal?

wista-project does not host video content.
It experiments with access, metadata, and playback mechanisms.

Users are responsible for complying with local laws and platform terms.

---

## Is this a YouTube scraper?

No.

WISTA explores modular access using existing open-source tools.
It does not attempt to bypass paywalls, DRM, or authentication systems.

---

## Is this a piracy tool?

No.

The project focuses on architecture and experimentation,
not redistribution of copyrighted content.

---

## Does WISTA use the official YouTube API?

No.

Search and metadata access are intentionally decoupled from official APIs
to avoid platform lock-in.

---

## Why use yt-dlp?

yt-dlp is a well-maintained open-source project
that demonstrates how access layers can remain adaptable.

It is used experimentally, not as a hidden dependency.

---

## Is this anti-YouTube?

No.

This project is not ideological opposition.
It is architectural exploration.
