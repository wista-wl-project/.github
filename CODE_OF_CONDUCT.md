# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

---

## Our Pledge

We pledge to make participation in this project a harassment-free experience
for everyone, regardless of background or identity.

---

## Our Standards

Examples of positive behavior:
- Respectful communication
- Constructive criticism
- Open-minded discussion

Unacceptable behavior includes:
- Harassment or discrimination
- Trolling or personal attacks
- Publishing private information

---

## Enforcement

Project maintainers are responsible for enforcing this code.
